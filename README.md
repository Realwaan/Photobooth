## Components

### Landing Component
Animated landing page with draggable photo strips, gradient backgrounds, and call-to-action.

**Features:**
- Draggable photo strip cards with mouse tracking
- Animated gradient overlays and floating blobs
- Staggered entrance animations
- Interactive hover effects

### Camera Component
Real-time camera feed with filters and capture controls.

**Features:**
- MediaDevices API integration with error handling
- 8 live filters with CSS filter effects
- Front/back camera switching (mobile)
- Countdown timer and instant capture
- Debug panel with logs and status info
- Flash effect on photo capture

### Gallery Component
Photo management with preview and deletion.

**Features:**
- Grid layout of captured photos
- Lightbox modal for full-size preview
- Individual photo deletion
- Timestamp display
- Empty state messaging

### LayoutSelector Component
Visual layout selection with previews.

**Features:**
- 7 layout options with visual representations
- Active state highlighting
- Responsive grid display
- Photo count indicators

### Customization Component
Photostrip personalization controls.

**Features:**
- Frame color selection (5 colors + rainbow)
- Photo shape options (rounded, square, circle)
- Sticker selection (hearts, stars, flowers, none)
- Logo text input (custom branding)
- Date/Time toggle options
- Download and retake buttons

## Customization

### Adding New Filters

Edit `src/components/Camera.jsx`:

```javascript
const filters = [
  { name: 'your-filter', label: 'Your Filter', class: 'your-css-class', icon: 'üé®' },
  // ... existing filters
]

const getFilterStyle = (filter) => {
  switch (filter) {
    case 'your-filter': return 'your-css-filter(value)'
    // ... existing cases
  }
}
```

### Adding New Layouts

Edit `src/components/LayoutSelector.jsx` and `src/App.jsx`:

```javascript
// In App.jsx
const layoutRequirements = {
  // ... existing layouts
  yourLayout: 5, // number of photos required
}

// In LayoutSelector.jsx - Add new layout option
```

### Customizing Colors

Edit `tailwind.config.js` to modify the color scheme:

```javascript
theme: {
  extend: {
    colors: {
      // Add your custom colors
    }
  }
}
```

### Adding Animations

Add new keyframes in `tailwind.config.js`:

```javascript
keyframes: {
  'your-animation': {
    '0%': { /* start state */ },
    '100%': { /* end state */ }
  }
}
```

```

```bash

```bash
